## Setup

Download `hugo`:

    wget -O - https://github.com/gohugoio/hugo/releases/download/v0.112.3/hugo_extended_0.112.3_linux-amd64.tar.gz | tar xz hugo

Install `hugo`:

    mv hugo /usr/local/bin

## Running locally

    hugo server
