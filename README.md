## Jekyll Compose Workflow
- Make a new draft: `bundle exec jekyll draft <title>`
- Write that content baby
- Move draft to posts when ready: `bundle exe jekyll publish _drafts/<filename> --timestamp-format "%Y-%m-%d %H:%M:%S %z"`
